# React + Redux + React-Router 学习
## 安装
在项目根目录下运行 `npm install` 安装依赖

## 运行
**第一步：执行 `npm start` 命令**

```bash
npm start
```
**第二步：开启 node API 代理服务**

```bash
// 在根目录下，进入node-proxy 文件夹
cd node-proxy
// 开启 node 代理 API 服务
node index.js
```
浏览器将自动打开`localhost:3000/#/`，便可以访问应用。
